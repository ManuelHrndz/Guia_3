Estilos embebidos en las etiquetas HTML

Usar estilos CSS embebidos en las etiquets HTML es el peor y el menos utilizado de los metodos para agregrar reglas de estilos. No es nada recomendable por la dificultad que implica realizar un cambio posterior.

Esta forma de inlcuir CSS directamente en los elementos HTML solamente se utiliza en determinadas situaciones en las que se debe incluir un estilo muy especifico para un solo elemento en concreto.

Para insertar un estilo CSS en un elemento se utiliza la propiedad style de HTML, que en ningun momento se debe confundir con la etiqueta <style> que se introduce en la cabecera del documento.
